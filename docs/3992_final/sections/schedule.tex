\section{Schedule}
\label{sec:schedule}

Each milestone below is listed in the format: \textbf{[target date]} milestone name (responsible party): deliverable description. See the Gantt chart in Table \ref{tab:gantt} for a visual representation of the schedule.

\begin{itemize}
    \item \textbf{[Jun 2026]} Hull Complete (Harrison LeTourneau): Fiberglass hull with mounting hardware, deck, and static float test passed
    \item \textbf{[Jun 2026]} Keel Fabricated (Corbin Barney, Jared Pratt): Fin keel with steel-filled bulb, carbon fiber wrap, and mounting hardware
    \item \textbf{[Jun 2026]} Rudder Fabricated (Corbin Barney, Jared Pratt): Rudder shaft with bearings, mounting plates, and servo integration
    \item \textbf{[Jul 2026]} Wingsail Fabricated (Charbel Salloum, Connor Stevens): Fiberglass-covered foam wingsail with servo-driven tail linkage
    \item \textbf{[Jul 2026]} Complete Boat Assembly (ALAN Team): Integrated hull, keel, rudder, and sail with verified COE/CLR alignment and center of mass testing.
    \item \textbf{[Aug 2026]} Servo Control (Sensor Team): Closed-loop PWM control of rudder and sail servos under load
    \item \textbf{[Sep 2026]} LoRa Communication (LoRa Team): Robust bidirectional link with error checking between vessel and ground station
    \item \textbf{[Sep 2026]} Sensor Integration (Sensor Team): GPS, IMU, magnetometer, and wind vane integrated into firmware with sensor fusion
    \item \textbf{[Sep 2026]} RC Sailboat (ALAN Team): A functioning RC sailboat that can be controlled by an operator over LoRa
    \item \textbf{[Oct 2026]} MCU Firmware (Embedded Team): Optimized dual-core FreeRTOS with CM7 control loops and CM4 communication
    \item \textbf{[Oct 2026]} Positioning Subsystem (Corbin Barney, Harrison LeTourneau): Sensor fusion algorithm with GPS/IMU dead-reckoning
    \item \textbf{[Oct 2026]} Wingsail Software (Connor Stevens, Charbel Salloum): PID control loop for tail servo responding to wind conditions
    \item \textbf{[Oct 2026]} Ground Station Interface (Harrison LeTourneau, Jared Pratt): GUI with real-time telemetry, map display, and waypoint input
    \item \textbf{[Nov 2026]} Path Planning (Charbel Salloum, Jared Pratt): Waypoint navigation with tacking/jibing state machine
    \item \textbf{[Nov 2026]} Emergency Recovery (Corbin Barney): Return-to-home on communication loss; low-battery sail furling, and capsize auto-protection
    \item \textbf{[Dec 2026]} Autonomous Sailboat (ALAN Team): Full autonomous waypoint navigation demonstration on open water
\end{itemize}
